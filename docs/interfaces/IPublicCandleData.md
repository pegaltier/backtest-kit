# IPublicCandleData

## Properties

### timestamp

```ts
timestamp: number
```

### open

```ts
open: number
```

Opening price at candle start

### high

```ts
high: number
```

Highest price during candle period

### low

```ts
low: number
```

Lowest price during candle period

### close

```ts
close: number
```

Closing price at candle end

### volume

```ts
volume: number
```

Trading volume during candle period
